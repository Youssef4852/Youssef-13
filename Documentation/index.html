6. Customization Guide

You can easily customize colors by editing :root variables in style.css: 

:root {
  --main-color: #8aed19; /* Change this to your brand color */
}

All icons come from Boxicons. Example: <i class="bx bx-user"></i>

9. How to Edit and Customize the Template

This section explains how to change the text, images, and colors in your template. All edits can be made easily using any code editor such as Visual Studio Code or Sublime Text.

1️⃣ Changing Text Content

All text content is written directly inside the HTML file. For example, to change the main heading in the home section: 


<h2>Navigating the digital <span>landscape</span> for success</h2>

Simply edit it to your own text:

<h2>Build Your Business With Modern Solutions</h2>

Save the file and refresh your browser — the new text will appear immediately.

---

2️⃣ Changing Images

All images are located inside the assets/images/ folder. To replace any image, keep the same filename or update the image path in index.html. 

<img src="assets/images/portfolio-1.jpg" alt="" />

Example: if you upload a new image named new-project.jpg, just replace the line with: 

<img src="assets/images/new-project.jpg" alt="My Project" />
---

3️⃣ Changing Colors

You can easily update the main color of the template by editing the CSS variable in style.css at the very top: 

:root {
  --main-color: #8aed19;
}

Just replace #8aed19 with your preferred color code, for example: 

:root {
  --main-color: #ff5722;
}

This will automatically update all buttons, highlights, and accent colors throughout the site.

---

4️⃣ Changing Section Titles

Each section (About, Services, Portfolio, Contact) has a title inside: 

<div class="main-heading">
  <h2>About</h2>
</div>

Change the word About to any name you want (e.g. "Who We Are").

---

5️⃣ Editing Footer Information

To change the footer text or social links, open the bottom part of index.html: 

<p class="reserved">
  &copy; <span>Youssef 13</span> All Right Reserved
</p>

You can replace "Youssef 13" with your name or brand, and add new icons using Boxicons: 

<i class="bx bxl-facebook"></i>
<i class="bx bxl-twitter"></i>
<i class="bx bxl-instagram"></i>
---

6️⃣ Editing the Contact Information

Scroll to the Contact section inside index.html and update your details: 

<p class="desc">+201234567891</p>
<p class="desc">business@youssef.com</p>
<p class="desc">Street 5, New York</p>

Replace these with your actual phone, email, and address.

---

7️⃣ JavaScript Behavior

The template uses script.js for two main features:

  • Header scroll effect – adds .scrolled class after 100px scroll.
  • Portfolio filtering – show/hide projects by category.

You can modify the script in assets/js/script.js to adjust behavior (e.g., change scroll threshold or add new filters).

---

💡 Tip:

Always make a backup of your files before making changes. After editing, refresh your browser or clear cache (Ctrl + Shift + R) to see updates.
